About the Server

Welcome to ZCPM Zombieland — a community-run 7 Days to Die dedicated server focused on survival, crafting, and cooperative zombie-slaying. Build your fortress and survive the horde!

⚙️

Server Details

  • Game: 7 Days to Die
  • Mode: Survival
  • PvP: Optional / Friendly
  • Mods: ZCPM (Zombie & Crafting Pack Mod)
🧟

Features

⚒ Custom Mods 👥 Active Community 🎉 Regular Events 🖥 Dedicated Hardware ☣ Zombie Hordes
🎮

How to Join

  1. Launch 7 Days to Die from your Steam library
  2. Navigate to Play → Connect to IP
  3. Enter the server address provided by the admin
  4. Enjoy the apocalypse!
