About the Server

How to Join

  1. Launch 7 Days to Die from your Steam library
  2. Navigate to Play → Connect to IP
  3. Enter the server address provided by the admin
  4. Enjoy the apocalypse!

📦 How to Install Mods

🖥️ Server-Side Installation

  1. Connect to your dedicated server via FTP / SFTP or your hosting panel's file manager.
  2. Navigate to the server root folder — the same directory that contains 7DaysToDieServer or startserver.sh.
  3. Open (or create) the Mods folder in the server root.
  4. Extract each mod ZIP file so that every mod's own folder lands directly inside Mods/ — e.g., Mods/TFP_WebServer/, Mods/TFP_MapRendering/.
  5. Restart the dedicated server. Check the server log for "Loaded mod: …" lines to confirm each mod loaded.
  6. For the web dashboard (TFP_WebServer), make sure WebDashboardEnabled=true and EnableMapRendering=true in your serverconfig.xml.

💻 Client-Side Installation (Player Guide)

  1. Download all mods listed under Client & Server above. Ask the server admin for the download links.
  2. Locate your 7 Days to Die installation folder. In Steam: right-click the game → Manage → Browse local files.
  3. Open (or create) the Mods folder inside the game directory.
  4. Extract each mod ZIP so that every mod's own folder lands directly inside Mods/ — e.g., Mods/0_TFP_Harmony/, Mods/AGF-V2-HUDPlus-1Main-v5.3.0/.
  5. Install 0_TFP_Harmony.zip first — it is a required framework for the other mods.
  6. If your server has Easy Anti-Cheat (EAC) disabled, launch the game via the "Play without EAC" option or the 7DaysToDie.exe launcher.
  7. Join the server. Press F1 and check the console log for "Loading mod …" to confirm your mods are active.
