About the Server

Welcome to ZCPM Zombieland – a community-run 7 Days to Die dedicated server focused on survival, crafting, and cooperative zombie-slaying fun. Join us, build your fortress, and survive the horde!

Server Details

  • Game: 7 Days to Die
  • Mode: Survival
  • PvP: Optional / Friendly
  • Mods: ZCPM (Zombie & Crafting Pack Mod)

Features

Custom Mods Active Community Regular Events Dedicated Hardware Zombie Hordes

How to Join

  • Launch 7 Days to Die and go to Play → Connect to IP
  • Enter the server address provided by the admin
  • Enjoy the apocalypse!
